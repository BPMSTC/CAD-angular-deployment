# 07 - Instructor Walkthrough (One Class Session)

## Suggested 60-90 minute pacing

1. **10 min** - Framework motivation (vanilla JS pain points)
2. **10 min** - Angular setup + CLI mental model
3. **15 min** - Architecture tour (components + service + model)
4. **15 min** - Data binding live coding
5. **15 min** - Signals live update demonstration
6. **10 min** - TypeScript recap + Q&A

## Student activity ideas

- add priority to `TaskMetadata`
- add filter buttons (todo/in-progress/done)
- add delete task button
- connect to local storage in `TaskService`

## Agile and team collaboration tie-in

The Team Task Tracker connects naturally to sprint reviews, workload visibility, and collaborative planning conversations.
