# Angular 21 Learning Hub

A focused Angular 21 educational SPA.

## Covered topics

- Angular philosophy + SPA model
- CLI + project structure
- Components + data binding
- Directives + control flow
- Services + Dependency Injection
- Routing
- HTTP + Observables
- Signals + modern reactivity

## Run locally

```bash
cd angular21-guide
npm install
npm start
```
